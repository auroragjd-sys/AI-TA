# 页面样式规范统一计划

## 问题分析
经过扫描发现，`src/App.tsx` 中存在以下样式不一致问题：
1.  **字体不统一**：除 Hero 和 CTA 区域外，大部分标题（`h2`, `h3`, `CardTitle`）缺失 `font-rounded-chinese` 类，导致圆体风格未贯穿全站。
2.  **阴影与边框混乱**：存在大量魔术数值（Magic Numbers）的边框颜色（如 `#ead6c5`, `#e8d4c4`, `#e8d5c5` 等），且卡片阴影使用不规范（有的无阴影，有的使用 `shadow-soft-lg`）。
3.  **按钮风格差异**：页脚按钮缺失阴影和圆角规范，与其他按钮风格脱节。
4.  **背景透明度不一致**：卡片背景透明度在 `80%` 和 `85%` 之间摇摆。

## 修改目标
统一全站的字体、阴影、边框颜色和卡片风格，建立一致的视觉体验。

## 具体步骤

### 1. 全局字体统一
*   **文件**: `src/App.tsx`
*   **操作**:
    *   为所有 `h2` 标签添加 `font-rounded-chinese` 类。
    *   为所有 `CardTitle` 组件添加 `font-rounded-chinese` 类。
    *   为页脚的 `h3` 标签添加 `font-rounded-chinese` 类。

### 2. 规范化卡片样式
*   **文件**: `src/App.tsx`
*   **操作**:
    *   **边框颜色**: 统一浅色边框为 `#e6ccb3` (与 CTA 输入框一致) 或 `#ebd8ca` (较常用值)，消除细微色差。
    *   **背景透明度**: 统一卡片背景为 `bg-white/80` 或 `bg-white/90`，保持一致的通透感。
    *   **阴影**:
        *   主要展示卡片（如 Features, Scenes, Trace）：添加 `shadow-soft` 或 `shadow-soft-lg`。
        *   次要卡片（如 Pain points）：保持 `hover:shadow-soft-lg` 交互。

### 3. 统一按钮风格
*   **文件**: `src/App.tsx`
*   **操作**:
    *   **页脚按钮**:
        *   添加 `shadow-sm` 或 `shadow-md`。
        *   确保使用 `rounded-full`。
        *   边框颜色统一为 `#e6ccb3`。

### 4. 细节微调
*   **文件**: `src/App.tsx`
*   **操作**:
    *   检查并统一 Section 的 `py-24` 间距。
    *   确保所有图标容器的圆角和背景色风格一致。

## 预期效果
页面将呈现统一的圆体字体风格，卡片和按钮拥有规范的边框和阴影，整体视觉更加协调、专业。
